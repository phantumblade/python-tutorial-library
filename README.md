</div>

```bash
# Clona il repository
git clone https://github.com/phantumblade/python-tutorial-library.git
cd python-tutorial-library

# Apri nel browser
open index.html

# Oppure avvia un server locale
python -m http.server 8000    # Python 3
python -m SimpleHTTPServer 8000    # Python 2
npx http-server    # Node.js

# Visita http://localhost:8000
```

### <img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/git/git-original.svg" width="26" height="26" alt="Workflow"/> Workflow di Sviluppo

```bash
# Setup iniziale
git clone https://github.com/phantumblade/python-tutorial-library.git
cd python-tutorial-library

# Crea un branch per le tue modifiche
git checkout -b feature/miglioramento-ui

# Modifica i file
# Edita tutorial, aggiungi esempi, migliora CSS/JS

# Testa le modifiche
python -m http.server 8000

# Commit e push
git add .
git commit -m "✨ Aggiunge nuovo tutorial su decoratori"
git push origin feature/miglioramento-ui

# Crea Pull Request su GitHub
```
